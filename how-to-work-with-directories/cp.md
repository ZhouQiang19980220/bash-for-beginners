# `cp`

`cp`命令是 Linux 系统中用于复制文件和目录的命令。

## 语法
```sh
cp [source] [destination]
```

## 示例
```sh
cp project02/file.txt project01/file.txt
```
- `cp`：复制文件或目录的命令。
- `project02/file.txt`：源文件路径。
- `project01/file.txt`：目标文件路径，复制后的文件名为 file.txt。
- 这个命令将 project02 目录中的 file.txt 文件复制到 project01 目录，并命名为 file.txt。如果 project01/file.txt 已经存在，它将被覆盖。
